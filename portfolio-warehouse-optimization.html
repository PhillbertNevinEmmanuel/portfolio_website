Data Analysis Project

Warehouse Optimization and Visualization using Python Pandas and Plotly Express

Scroll

Project Description

Developed a warehouse optimization workbook using Python tools like Pandas, and Plotly Express to improve product allocation across plants and branches by prioritizing items with the highest nett margin, enforcing demand constraints, and applying an ascending plan sequence for efficient distribution. The system processed and analyzed margin, cost, and nett margin data to generate a data-driven allocation strategy, with interactive visualizations enabling stakeholders to identify high-value products and make informed distribution decisions.
