Tools Used:

Designed and implemented a data solution that enabled stakeholders to track and manage sales order performance by principal, brand, and item. Sourced data from sales and customer databases, developed APIs from SQL queries, and integrated the results into a Qlik Sense dashboard via REST API. The dashboard monitored sales orders by both volume and revenue, and analyzed average basket size per store each month.

This provided consistent oversight of sales order trends, delivered actionable insights on principal and brand performance by location, and supported data-driven decisions on which principals offered the highest distribution potential.
